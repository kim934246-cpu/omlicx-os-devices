## What This Repo Contains

This repository is **the device-specific core for the Pixel 3a**. It includes everything needed to make OmlexOS run flawlessly on this phone:

- **Device Tree & Kernel Configs**: Optimized for Pixel 3a hardware.  
- **Drivers & Modules**: Audio, display, sensors, camera, Wi-Fi, Bluetooth—all prepped and organized for rapid development.  
- **Halium & Build Scripts**: Smooth integration with Halium, making Ubuntu ARM and Linux apps play nicely.  
- **Flash & Environment Setup**: Scripts to make building and testing on the Pixel 3a seamless.  
- **Documentation**: Step-by-step guides to set up, build, and flash OmlexOS on your Pixel 3a.

This is **your playground**. A place where you can drop in drivers, tweak configs, experiment with kernels, and bring your vision of a radical mobile OS to life.
